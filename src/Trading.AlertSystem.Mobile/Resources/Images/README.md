# 图片资源

此目录存放应用图片资源。

## 需要的图标（可选）

如果要在侧边栏显示图标，需要添加以下图片：
- monitor_icon.png (监控状态图标)
- alert_icon.png (告警规则图标)
- history_icon.png (历史记录图标)
- ema_icon.png (EMA配置图标)
- settings_icon.png (设置图标)
- add_icon.png (添加按钮图标)

建议尺寸: 24x24 或 32x32 像素

可以从以下网站下载免费图标:
- https://fonts.google.com/icons
- https://icons8.com
- https://www.flaticon.com

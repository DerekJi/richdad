# 策略性能分析器

## 目的

专门分析2024年黄金Pin Bar策略表现不佳的原因，并提供优化建议。

## 问题背景

根据回测结果，2024年3月至12月期间：
- **胜率**: 27.04% (43胜/116负)
- **总收益**: -$8,500
- **收益率**: -8.50%
- **实际盈亏比**: 0.93 (低于策略设定的1.5)

## 分析维度

### 1. 总体表现分析
- 胜率统计
- 盈亏分布
- 实际盈亏比计算
- 问题识别（胜率低、总体亏损、盈亏比倒挂）

### 2. 亏损交易模式分析
- 止损触发频率
- 持仓时间分布
- 多空方向偏向
- 月度亏损分布

### 3. 时间模式分析
- UTC小时级别表现
- 星期级别表现
- 识别表现较差的时间段

### 4. 市场环境分析
- 2024年黄金市场特征
- 策略在不同市场环境下的表现
- 问题诊断
- 优化方向建议

## 运行方式

```bash
cd src/Trading.Strategy.Analyzer
dotnet run
```

## 输出内容

程序会输出：
1. 总体统计数据
2. 盈亏交易对比
3. 时间分布分析
4. 问题诊断结论
5. 优化方向建议

## 已识别的问题

1. **胜率过低**：< 30%，说明策略信号质量不高
2. **总体亏损**：策略在2024年市场环境下失效
3. **盈亏比倒挂**：实际盈亏比0.93 < 1.0，平均亏损大于平均盈利

## 可能原因

1. Pin Bar策略在强趋势市中表现不佳
2. 2024年黄金处于上涨趋势，策略更适合震荡市
3. 止损设置在高波动环境下容易被触发
4. EMA200在快速趋势中滞后性明显

## 建议优化方向

- [ ] 添加趋势强度过滤（ADX指标）
- [ ] 在强趋势市暂停交易或调整参数
- [ ] 优化止损策略（移动止损或时间止损）
- [ ] 添加波动率过滤（ATR阈值）
- [ ] 实现自适应参数（根据市场环境调整）
- [ ] 考虑只在回调或震荡时交易

## 下一步

根据分析结果，在 `strategy/improve-2024-performance` 分支实施具体优化方案。
